# Favicon Creation Instructions

## To Create Castello Coffee Favicons:

1. Open `public/4bw11l17jtsb1.jpg` in an image editor (Photoshop, GIMP, etc.)

2. Create the following sizes:
   - **favicon.ico**: 64x64 px (multi-resolution .ico file)
   - **favicon.png**: 32x32 px (PNG format)

3. Ensure the logo is centered and has a white or transparent background

4. Save both files in the `public` folder:
   - `public/favicon.ico`
   - `public/favicon.png`
